Wrathful Hedgehog

About

Wrathful Hedgehog was our first game project at FutureGames.

The game was built with C# and Unity together with a team of 14 people assigned by FutureGames.

Info

Role: Character & Gun Systems Programmer

Team Size: 14

Time: 2 Months

Engine: Unity

Introduction

In this project I took the responsibility of programming the game's Movement and Weapon system We used Perforce desktop as source control, Miro for prototyping, Jira for project management. I mainly put lot of focus on the Weapon system but I also worked a bit on making the cave shrink, health, and the UI implementation.

Systems

The Ammo depleting for the weapons was fairly easy, when I got to shooting I had to use unity's raytraying method to make sure a bullet point in the screen would hit the enemies.

Character Controller

The Character Controller was used to make the character move, at first I used rigidbody, but I noticed problems arise with the movement, you would slide like you were on ice, and the rigibody didn't function well with how character should move to put it simply.

Lessons learned:

Working on this project with other people taught me very valuable lessons in game dev. I learned that communication is key. The way we articulate or speak to one another is so very important, for success in a Branch this different from the rest, the coders have to keep up with the artists, the artist the UXD, the QA have to talk to the programmers, vice versa in all different ways.
