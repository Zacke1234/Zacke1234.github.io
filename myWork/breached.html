Introduction

In this project I worked on the gun systems and the UI implementation of it.

Gun systems

Working on the gun system I started with a base in c# and later continued making the more complex features in blueprints, the Cryofreeze gun was quite the hassle to make though, I had to freeze enemies and keep count of the ammo, reload time, etc. The burst gun was also very unique in that I had to make a variable for how many bullets it would burst out, and if it should be a automatic burst fire weapon, there was alot of back and forth with the designers on how the weapons should work. At the start the Cryofreeze gun was gonna have something called the "chaos system" which lets you simulate physics on objects that break apart like glass, and we wanted that for the freeze gun, but that unfortunately got scrapped.
