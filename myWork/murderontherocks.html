Introduction

In this project I worked on the camera switching, UI & Sounds implementation and a bit on the dialouge system.

Dialogue system

Getting the dialouge system to work was quite difficult, it was hard to know how to make it via blueprints, there wasn't much great information about it online. Looking back on it, I think I should have switched tasks much sooner when it came to this, it drained alot of my energy in this project trying to get this to work, so I passed it on another programmers duty to do it, and they did it very flawlessly. Of course they probably had some of my base of the dialouge system but they did the rest. Getting the a history function to work was nearly impossible for me, I made a basic Yes or no option and linear text but the rest was up to the other programmers.


Role: Systems Programmer

Murder on the rocks was our third game project at FutureGames.

The game was built with C# & Blueprints on Unreal engine together with a team of 12 people assigned by FutureGames.

Programming
